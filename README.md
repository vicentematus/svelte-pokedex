# SvelteKit PokeDex

## Developing / Desarrollo

First install pnpm as the package manager

```bash
npm install -g pnpm
```

Then install the packages

```bash
pnpm install 
```

Run the dev server

```bash
pnpm run dev 
```

## Production Enviroment 

To create a production version of your app:

```bash
pnpm run build

```
